# StudyDesk

Your college work — organized, on time, done right.

**Live demo:** [cardyop.github.io/studydesk](https://cardyop.github.io/studydesk/)

## About

StudyDesk is a simple web app for tracking college assignments and tasks. Add what needs to be done, set a subject, due date, due time, and priority, then track everything from one clean dashboard.

## Features

- Add new tasks with subject/course, due date, due time, and priority level
- Priority levels: Low, Medium, High
- Filter tasks by status: All, Pending, Done
- Quick filters for High priority and Overdue tasks
- Task counter to see how much is on your plate

## Getting Started

### Live Version

Just open the live demo link above — no installation needed.

### Run Locally

1. Clone the repository
   ```bash
   git clone https://github.com/cardyop/studydesk.git
   ```
2. Navigate into the project folder
   ```bash
   cd studydesk
   ```
3. Open `index.html` in your browser

## Usage

1. Enter the task in the "What's the work?" field
2. Fill in the subject/course, due date, due time, and priority
3. Click "＋ Add to Desk" to save the task
4. Use the filter tabs (All, Pending, Done, High, Overdue) to view your tasks
